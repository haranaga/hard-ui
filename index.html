About HARD-UI

HARD-UI is a simple & lightweight CSS without any Javascript.
No roundness. No softness. No complexness.
Yes, responsive framework.

Get started

  1. Download hard-ui.js or hard-ui.min.js.
  2. Add html tag.

<link rel="stylesheet" href="/css/hard-ui.css">
