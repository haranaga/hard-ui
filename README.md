# hard-ui
HARD-UI is a Web ui flamework.
* without Javascript
* responsive
* lightweight

## Status
under construction

## Get started
1. Download hard-ui.css or hard-ui.min.css.
2. Add html tag.

```
<link rel="stylesheet" href="/path/to/hard-ui.css">
```
